# How to run progressive web apps on the Flx1s

1. Install the PWA addon for firefox.
https://addons.mozilla.org/de/firefox/addon/pwas-for-firefox/
2. Follow the instruction
3. After installation to Mobile Settings App (Do not close firefox)
4. Go to Compositor and enable down scale for nextcloud-desktop
5. Now you can see the install PWA button on PWA compatible websites
