# Information
Nextcloud Desktop is currently broken when installed via flatpak (07.01.2026)
It is reccomended to install it via apt.

# Installation of Nextcloud Desktop

1. Install nextcloud-desktop
```bash
sudo apt install nextcloud-desktop
```
2. Go to Mobile Settings App (Do not close nextcloud-desktop)
3. Go to Compositor and enable down scale for nextcloud-desktop

It will create a folder in your Home directory called Nexcloud where all files are synced.
