# Installation of Brave Browser

1. Install curl
```bash
sudo apt install curl
```
2. Install brave via cli ( https://brave.com/linux/)
```bash
curl -fsS  https://dl.brave.com/install.sh  | sh
```
3. Open Brave in the terminal
```bash
brave-browser
```
4. Finish setup
5. Close brave
6. Go to the main menu and open brave via Icon
7. Go to Mobile Settings App (Do not close brave)
8. Go to Compositor and enable down scale for brave
